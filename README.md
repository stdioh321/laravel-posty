# Posty
Testing Laravel 8.

# Setup
```shell
git clone https://github.com/stdioh321/laravel-posty.git
cd laravel-posty
composer install
npm install
```

# Run
```shell
php artisan serve
```
**Open your browser at: http://localhost:8000**

# Run with Docker
```shell
bash ./startup_docker.sh
```

**Open your browser at: http://localhost:8000**
